记一次安装tplmap使用时的坑：

​ 该程序要用python2运行 ，但是他会提示我没有安装yaml (import error)

​ 首先看了二向箔的博客，说是要用pip install -r requirements提供依赖环境，这个好理解。

​ 尝试之后，提示我error：没有这个文件。

​ 于是解决这个文件的问题，全局搜索没有找到，后来找到一篇博客，说是要先创建一下：

pip freeze > requirements.txt

然后再执行pip install -r requirements.txt

​ 解决了这个问题，再来解决yaml的问题。

​ 得用pip安装；但是我py2没有pip，所以又得下载。注意，要在py2下下载pip。很多系统默认只给py3安装pip。
